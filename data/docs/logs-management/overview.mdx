## Key Capabilities

- **[Flexible Ingestion](https://signoz.io/docs/logs-management/send-logs-to-signoz/)** — Collect logs from files, stdout, FluentBit/FluentD/Logstash, OpenTelemetry SDKs, HTTP endpoints, and cloud services.
- **[Logs Explorer](https://signoz.io/docs/product-features/logs-explorer/)** — Search, filter, and analyze logs with List, Time Series, and Table views. Stream live logs in real time.
- **[Query Builder](https://signoz.io/docs/userguide/logs_query_builder/)** — Build advanced queries with aggregations, grouping, and multi-query comparisons.
- **[Log Pipelines](https://signoz.io/docs/logs-pipelines/introduction/)** — Parse, transform, and enrich logs before storage using SigNoz's functional pipeline builder.
- **[Alerts](https://signoz.io/docs/alerts-management/log-based-alerts/)** — Set up alerts on log patterns, counts, or attribute values.
- **[Unified Context](https://signoz.io/docs/traces-management/guides/correlate-traces-and-logs/)** — Correlate logs with traces and metrics for faster root-cause analysis.

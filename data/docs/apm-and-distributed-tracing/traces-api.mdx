The SigNoz Traces API lets you search and aggregate trace data programmatically.

## Endpoint

```
POST https://{URL}/api/v5/query_range
```

Replace `{URL}` with your SigNoz instance URL (e.g., `your-instance.signoz.cloud`).

## Authentication

Generate an API key using a [Service Account](https://signoz.io/docs/manage/administrator-guide/iam/service-accounts/). Pass it in your request header:

```bash
SIGNOZ-API-KEY: {YOUR_API_KEY}
```

## Operations

- **[Search traces](https://signoz.io/docs/traces-management/trace-api/search-traces)** — filter traces by service, duration, error status, or any span attribute.
- **[Aggregate traces](https://signoz.io/docs/traces-management/trace-api/aggregate-traces)** — compute metrics (count, avg, p99, etc.) over filtered trace data.
- **[Payload model](https://signoz.io/docs/traces-management/trace-api/payload-model)** — request and response schema reference.

Use public sharing when you want to make a dashboard available with a link outside the authenticated SigNoz UI.

<Figure
  src="/img/docs/dashboards/overview/share-page-link.webp"
  alt="Share page link option in SigNoz dashboard"
  caption="Share the current page link with your team members."
/>

## Enable Public Sharing

1. From the sidebar, choose **Dashboards** and select the respective dashboard you want to share publicly.
2. Open the configuration drawer for that dashboard by clicking the **Configure** icon.
3. In the configuration drawer, select **Publish**.
4. To allow viewers to change the time range on the public dashboard, toggle the **Enable time range** option.
5. Choose the **Default time range** that the public dashboard should use.

<Admonition type="info">
  **Important**: If the time range option is disabled, viewers will only be able to see the
  dashboard for the specified default time range.
</Admonition>

6. Click the **Publish Dashboard** button to enable public sharing.
7. A publicly shareable link will be generated, which you can use to provide public access to the dashboard.

<Figure
  src="/img/docs/dashboards/public-sharing/enable-public-sharing.webp"
  alt="Public sharing configuration in SigNoz"
  caption="Once published, copy the public dashboard URL or open it directly. Dashboard variables won't work in public dashboards."
/>

## Disable Public Sharing

1. From the sidebar, choose **Dashboards** and select the respective dashboard for which you want to revoke public access.
2. Open the configuration drawer for that dashboard by clicking the **Configure** icon.
3. Navigate to the dashboard configuration section and select **Publish**.
4. To disable public sharing for the dashboard, click the **Unpublish dashboard** button.

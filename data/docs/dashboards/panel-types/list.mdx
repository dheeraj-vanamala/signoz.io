## List Chart

A List chart is a plot chart that shows a list of values. This is particularly useful when you want to show a list of values in a single panel. A typical use case is to show the error logs or the traces in a single panel. The panel supports infinite scrolling and search.

### Data Formats

#### Supported signals

- Logs
- Traces

This panel type supports showing the logs lines and spans in a list.

### Configuration

The columns displayed in the list are configurable. You can add, remove, and reorder columns to show the attributes most relevant to your investigation.

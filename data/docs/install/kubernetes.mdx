Learn how to install SigNoz on Kubernetes with Helm.

<KeyPointCallout title="Deploy with Foundry" defaultCollapsed={false}>
<a href="https://github.com/SigNoz/foundry" target="_blank" rel="noopener noreferrer nofollow">Foundry</a> can generate Kubernetes manifests for SigNoz using either <a href="https://github.com/SigNoz/foundry/tree/main/docs/examples/kubernetes/helm" target="_blank" rel="noopener noreferrer nofollow">Helm casting</a> or <a href="https://github.com/SigNoz/foundry/tree/main/docs/examples/kubernetes/kustomize" target="_blank" rel="noopener noreferrer nofollow">Kustomize casting</a>. You provide a casting configuration file, and Foundry generates the Helm values or Kustomize manifests for your deployment.

For a full Kustomize-based deployment guide, see [Deploying to Kubernetes with Kustomize](https://signoz.io/docs/setup/kubernetes/kustomize/).
</KeyPointCallout>

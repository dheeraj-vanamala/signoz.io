# Create Feature Page

Use the `signoz-feature-page-builder` skill to create a new SigNoz feature page.

Follow the standard page architecture, shared components, and design patterns from existing feature pages.

Before starting, clarify:
1. Feature name and URL path
2. Key value propositions (3-5)
3. Target audience
4. Available product screenshots
5. Primary and secondary CTA destinations

Then build the page following the standard flow: Header → Feature Sections → CTA Banner → UsageBasedPricing → SigNozStats → CustomerStoriesSection.
